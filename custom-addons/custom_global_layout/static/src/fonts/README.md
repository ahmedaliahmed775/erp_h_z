# مجلد الخطوط — Cairo Font
# ============================================
#  wkhtmltopdf لا يستطيع تحميل الخطوط من Google Fonts عن طريق @import.
#  يجب تحميل ملفات الخط يدوياً ووضعها هنا.
#
#  رابط التحميل:
#  https://fonts.google.com/specimen/Cairo
#
#  حمّل الملفات التالية (بصيغة TTF) وضعها في هذا المجلد:
#  - Cairo-Regular.ttf
#  - Cairo-SemiBold.ttf
#  - Cairo-Bold.ttf
#  - Cairo-ExtraBold.ttf
#
#  أو من سطر الأوامر (بعد تحميل Google Fonts API key):
#  curl -L -o Cairo-Regular.ttf  "https://github.com/google/fonts/raw/main/ofl/cairo/static/Cairo-Regular.ttf"
#  curl -L -o Cairo-Bold.ttf     "https://github.com/google/fonts/raw/main/ofl/cairo/static/Cairo-Bold.ttf"
#  curl -L -o Cairo-SemiBold.ttf "https://github.com/google/fonts/raw/main/ofl/cairo/static/Cairo-SemiBold.ttf"
#  curl -L -o Cairo-ExtraBold.ttf "https://github.com/google/fonts/raw/main/ofl/cairo/static/Cairo-ExtraBold.ttf"
